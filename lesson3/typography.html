Typography

A serif font called Playfair Display will be used for the headlines. A sans serif font called Roboto will be used in the paragraphs and the navigation menu. Each font was selected from the Google Fonts website. The root font size is 16px which is the font size for the body of the web pages when a large viewport is used. The headlines font sizes are measured using the rem unit. The rem units for the font size are adjusted based on the media queries and the viewport size. The font sizes are adjusted to make the web pages responsize to the changes in viewport size. Accessibility was considered when determining fonts and font sizes.
